## Run Locally

Clone the project

```bash
  git clone https://github.com/JoelDeonDsouza/Front-End-Online-Bank-App
```

Go to the project directory


Start the React-App

```bash
  npm start
```

## Features

- Landing Page.
- Home Section with card and amount display.
- Navigation throught the screens.
- Dynamic component rendering.
- Card(virtual card) selection and add on.
- Add and see user accounts.
